# 数据字段说明

## 元素成分字段

建议包含以下元素列：

```text
Ni
Co
Cr
Al
Ti
Ta
W
Mo
Re
Nb
Hf
C
B
```

元素成分默认按百分比处理。

如果元素成分总和不等于 100，可以在特征工程阶段归一化。

## 工艺字段

```text
solution_temp_c
solution_time_h
aging_temp_c
aging_time_h
cooling_rate_c_s
```

## 测试条件字段

```text
test_temp_c
stress_mpa
```

## 回归目标字段

```text
yield_strength_mpa
tensile_strength_mpa
creep_log_life_h
oxidation_mass_gain_mg_cm2
```

## 分类目标字段

```text
creep_life_class
```

## 字段要求

真实数据中可以只包含部分目标列。运行脚本时指定的目标列必须存在。

例如只做蠕变寿命预测时，至少需要包含：

```text
Ni
Co
Cr
Al
Ti
Ta
W
Mo
Re
Nb
Hf
C
B
solution_temp_c
solution_time_h
aging_temp_c
aging_time_h
cooling_rate_c_s
test_temp_c
stress_mpa
creep_log_life_h
```
