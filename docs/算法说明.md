# 算法说明

本项目包含传统机器学习、聚类、降维、深度学习和模型解释相关方法。

## 传统机器学习

### 线性模型

```text
Linear Regression
Ridge
Lasso
ElasticNet
Logistic Regression
```

用途：

```text
Linear Regression     回归基线模型
Ridge                 带 L2 正则的线性回归
Lasso                 带 L1 正则的线性回归，可产生稀疏特征权重
ElasticNet            同时结合 L1 和 L2 正则
Logistic Regression   分类基线模型
```

### 邻近算法

```text
KNN Regressor
KNN Classifier
```

用途：

```text
适合做简单非参数基线
对特征尺度敏感
通常需要标准化
```

### 核方法

```text
SVR
SVM Classifier
```

用途：

```text
适合中小规模非线性数据
对标准化敏感
参数 C、gamma、epsilon 对结果影响明显
```

### 树模型

```text
Decision Tree
Random Forest
Extra Trees
Gradient Boosting Tree
XGBoost
```

用途：

```text
Decision Tree             可解释性较好，但容易过拟合
Random Forest             稳健的表格数据模型
Extra Trees               随机性更强的树集成模型
Gradient Boosting Tree    逐步提升的树模型
XGBoost                   强表格数据基线模型
```

## 聚类算法

```text
KMeans
DBSCAN
Agglomerative Clustering
```

用途：

```text
KMeans                    根据距离划分样本，需要指定簇数量
DBSCAN                    基于密度聚类，可识别噪声点
Agglomerative Clustering  层次聚类，可用于观察样本层级关系
```

## 降维算法

```text
PCA
t-SNE
```

用途：

```text
PCA       线性降维，可查看解释方差比例
t-SNE     非线性可视化方法，适合观察局部分布
```

## 深度学习

```text
MLP
CNN
RNN
LSTM
Transformer
```

用途：

```text
MLP          表格数据回归和分类
CNN          图像或二维特征图示例
RNN          序列数据示例
LSTM         序列数据示例，比普通 RNN 更适合长依赖
Transformer 序列建模示例
```

## 模型解释

```text
Feature Importance
Permutation Importance
SHAP
```

用途：

```text
Feature Importance       使用模型自带的重要性或线性系数
Permutation Importance   打乱特征后观察性能下降
SHAP                     解释单个特征对预测结果的贡献
```
