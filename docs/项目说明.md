# 项目说明

本项目用于构建高温合金表格数据的机器学习分析流程。

主要流程包括：

1. 数据读取
2. 数据检查
3. 特征工程
4. 传统机器学习建模
5. 聚类分析
6. 降维分析
7. 深度学习建模
8. 模型解释
9. 结果汇总

项目默认生成合成数据，便于直接运行。后续也可以替换为外部 CSV 或 Excel 表格数据。

## 主要数据流

```text
原始数据
  ↓
数据检查
  ↓
特征工程
  ↓
预处理流水线
  ↓
模型训练
  ↓
模型评估
  ↓
图表与报告输出
```

## 默认数据来源

默认配置中：

```yaml
data:
  source: synthetic
```

程序会读取或生成：

```text
data/processed/synthetic_superalloy.csv
```

## 外部数据来源

如需使用外部数据，可将表格放入：

```text
data/raw/
```

然后修改配置：

```yaml
data:
  source: external
  external_path: data/raw/my_superalloy_data.csv
```

也可以在运行脚本时传入路径，例如：

```bash
python scripts/run_traditional_ml.py --data-path data/raw/my_superalloy_data.csv --regression-target creep_log_life_h
```

## 输出目录

运行结果统一保存到：

```text
outputs/
```

其中：

```text
outputs/models/       保存模型文件
outputs/reports/      保存结果表和报告
outputs/figures/      保存图像
outputs/predictions/  保存预测结果
```
