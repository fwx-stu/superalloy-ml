# 评估指标说明

## 回归指标

### MAE

平均绝对误差。

```text
MAE = mean(|y_true - y_pred|)
```

特点：

```text
容易理解
与目标变量单位一致
对异常值不如 MSE 敏感
```

### MSE

均方误差。

```text
MSE = mean((y_true - y_pred)^2)
```

特点：

```text
会放大大误差
对异常值敏感
```

### RMSE

均方根误差。

```text
RMSE = sqrt(MSE)
```

特点：

```text
与目标变量单位一致
比 MAE 更关注大误差
```

### R2

决定系数。

```text
R2 = 1 - SS_res / SS_tot
```

特点：

```text
越接近 1 越好
可能为负数
适合评价整体拟合效果
```

### MAPE

平均绝对百分比误差。

```text
MAPE = mean(|(y_true - y_pred) / y_true|)
```

特点：

```text
便于理解相对误差
当真实值接近 0 时不稳定
```

### Max Error

最大误差。

特点：

```text
用于观察最差预测情况
对异常值非常敏感
```

## 分类指标

### Accuracy

准确率。

```text
Accuracy = correct_predictions / total_predictions
```

特点：

```text
直观
类别不平衡时可能误导
```

### Precision

查准率。

```text
Precision = TP / (TP + FP)
```

特点：

```text
关注预测为正的样本中有多少是真的正样本
```

### Recall

查全率。

```text
Recall = TP / (TP + FN)
```

特点：

```text
关注真实正样本中有多少被识别出来
```

### F1

Precision 和 Recall 的调和平均。

```text
F1 = 2 * Precision * Recall / (Precision + Recall)
```

特点：

```text
适合类别不平衡场景
比单独 Accuracy 更稳健
```

### Confusion Matrix

混淆矩阵用于查看不同类别之间的误判情况。

## 交叉验证

小样本表格数据建议重点查看交叉验证均值和标准差。

```text
均值反映平均性能
标准差反映稳定性
```

单次训练集测试集划分可能存在偶然性。
