# AI 辅助说明

本项目中的部分样板代码、测试代码和说明文本可以使用 AI 工具辅助生成。

使用 AI 工具后，仍需要人工检查以下内容：

```text
数据字段是否正确
目标列是否正确
特征工程是否符合数据含义
模型输入和输出维度是否正确
评估指标是否适合当前任务
结果是否存在明显异常
代码是否能完整运行
```

## 适合 AI 辅助的内容

```text
目录结构设计
重复性脚本生成
单元测试草稿
README 草稿
绘图函数草稿
配置文件模板
```

## 需要人工重点确认的内容

```text
材料字段含义
单位是否一致
目标列是否正确
特征工程是否合理
模型结果是否符合基本常识
异常值是否真实存在
数据是否存在泄漏
```

## 使用建议

AI 工具适合提高代码组织和初稿生成效率，但不能替代对数据、材料背景和模型结果的判断。

所有核心结果都应通过以下方式确认：

```text
运行脚本
查看输出表格
查看图像
检查日志
运行测试
人工复核关键公式和字段
```
