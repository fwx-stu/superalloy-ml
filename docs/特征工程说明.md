# 特征工程说明

本项目包含通用表格特征处理和高温合金相关特征工程。

## 成分相关特征

```text
composition_sum
gamma_prime_formers_total
refractory_total
oxidation_resistance_total
al_ti_ratio
ta_w_ratio
cr_al_ratio
co_ni_ratio
```

含义说明：

```text
composition_sum                 元素成分总和
gamma_prime_formers_total        γ′ 形成元素总量
refractory_total                 难熔元素总量
oxidation_resistance_total       抗氧化相关元素总量
al_ti_ratio                      Al 与 Ti 的比例
ta_w_ratio                       Ta 与 W 的比例
cr_al_ratio                      Cr 与 Al 的比例
co_ni_ratio                      Co 与 Ni 的比例
```

## 元素物性描述符

```text
avg_atomic_weight
avg_atomic_radius_pm
atomic_radius_mismatch
avg_electronegativity
electronegativity_mismatch
estimated_density_g_cm3
estimated_cost_index
mixing_entropy_r
```

含义说明：

```text
avg_atomic_weight                成分加权平均原子量
avg_atomic_radius_pm             成分加权平均原子半径
atomic_radius_mismatch           原子半径失配
avg_electronegativity            成分加权平均电负性
electronegativity_mismatch       电负性失配
estimated_density_g_cm3          估算密度
estimated_cost_index             估算成本指数
mixing_entropy_r                 理想混合熵的无量纲形式
```

## 工艺和测试条件特征

```text
solution_temp_k
aging_temp_k
test_temp_k
solution_time_log
aging_time_log
cooling_rate_log
solution_parameter
aging_parameter
temp_gap_solution_aging
test_temp_fraction_of_solution
```

含义说明：

```text
solution_temp_k                  固溶温度，单位 K
aging_temp_k                     时效温度，单位 K
test_temp_k                      测试温度，单位 K
solution_time_log                固溶时间的 log1p 变换
aging_time_log                   时效时间的 log1p 变换
cooling_rate_log                 冷却速率的 log1p 变换
solution_parameter               固溶温度与固溶时间组合参数
aging_parameter                  时效温度与时效时间组合参数
temp_gap_solution_aging          固溶温度和时效温度之差
test_temp_fraction_of_solution   测试温度与固溶温度的比例
```

## 交互特征

```text
stress_temp_interaction
refractory_temp_interaction
gamma_prime_aging_interaction
cr_oxidation_temp_interaction
```

含义说明：

```text
stress_temp_interaction          应力与测试温度交互项
refractory_temp_interaction      难熔元素与测试温度交互项
gamma_prime_aging_interaction    γ′ 形成元素与时效温度交互项
cr_oxidation_temp_interaction    Cr 与测试温度交互项
```

## 注意事项

这些特征用于机器学习流程演示和建模分析，不等同于完整的材料物理模型。

真实数据使用前建议检查：

```text
元素单位是否一致
温度单位是否一致
时间单位是否一致
应力单位是否一致
目标列是否经过必要转换
是否存在极端异常值
```
